# User Roles

## Primary Users

### Manual QA Engineer

Responsibilities

- Create projects
- Add requirements
- Generate AI-assisted test cases
- Manage testing documentation

---

### Automation QA Engineer

Responsibilities

- Generate test cases
- Review AI-generated outputs
- Use generated test cases for automation

---

## Secondary Users

### QA Lead

Responsibilities

- Review testing progress
- Monitor projects
- Review generated test cases

---

### Developer

Responsibilities

- Understand testing scenarios
- Review AI-generated test cases
- Collaborate with QA Engineers
